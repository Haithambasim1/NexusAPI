Professional API Services

A collection of production-ready API services built with Node.js and Express, designed to be deployed on RapidAPI.

Note: This documentation is for the local development server. When deployed to RapidAPI, the base URL and authentication will change.

Our APIs provide powerful, easy-to-use services for developers:

  • Email Validation API - Verify email addresses with comprehensive checks
  • Company Information API - Retrieve detailed company data based on domain name
  • Sentiment Analysis API - Analyze text content for emotional sentiment

Authentication

When deployed on RapidAPI, all API calls require RapidAPI authentication headers:

X-RapidAPI-Key: YOUR_RAPIDAPI_KEY
X-RapidAPI-Host: your-api.p.rapidapi.com

During local development, authentication can be disabled by setting NODE_ENV=development in your .env file.

Rate Limiting

To prevent abuse, our APIs implement rate limiting:

  • Standard tier: 100 requests per 15-minute window
  • Premium tier: 1000 requests per 15-minute window
  • Email validation: Maximum of 1000 checks per day (configurable)

When rate limits are exceeded, the API will respond with a 429 status code.

Error Handling

All endpoints follow a standard error response format:

{
  "success": false,
  "error": {
    "message": "Error message description",
    "code": "ERROR_CODE"
  }
}

Common error codes include:

  • INVALID_PARAMETERS - Missing or invalid request parameters
  • UNAUTHORIZED - Authentication failure
  • RATE_LIMIT_EXCEEDED - Too many requests
  • SERVER_ERROR - Internal server error

Email Validation API

Provides comprehensive email address verification with multiple checks.

GET /api/email/validate

Parameters

Name Type Description
emailrequired string The email address to validate

Response

{
  "success": true,
  "data": {
    "email": "user@example.com",
    "is_valid_format": true,
    "is_deliverable": true,
    "is_disposable": false,
    "suggestion": null,
    "mx_records": [
      { "exchange": "mx1.example.com", "priority": 10 },
      { "exchange": "mx2.example.com", "priority": 20 }
    ],
    "username": "user",
    "domain": "example.com",
    "score": 0.8,
    "checks": {
      "syntax": true,
      "domain_exists": true,
      "has_mx_records": true,
      "role_account": false,
      "disposable": false,
      "free_provider": false
    }
  }
}
{
  "success": false,
  "error": {
    "message": "Email parameter is required",
    "code": "INVALID_PARAMETERS"
  }
}

Company Information API

Retrieves detailed company information based on domain name.

GET /api/company/info

Parameters

Name Type Description
domainrequired string The company domain name

Response

{
  "success": true,
  "data": {
    "domain": "example.com",
    "company_name": "Example Inc.",
    "logo_url": "https://logo.clearbit.com/example.com",
    "industry": "Technology",
    "description": "A technology company focused on innovation.",
    "founded": 2010,
    "employees": 500,
    "location": {
      "country": "United States",
      "city": "San Francisco",
      "state": "CA"
    },
    "social_media": {
      "twitter": "https://twitter.com/example",
      "linkedin": "https://linkedin.com/company/example",
      "facebook": "https://facebook.com/example"
    }
  }
}
{
  "success": false,
  "error": {
    "message": "Domain parameter is required",
    "code": "INVALID_PARAMETERS"
  }
}

Sentiment Analysis API

Analyzes the sentiment of text content.

POST /api/ai/sentiment

Request Body

{
  "text": "I really love this product, it's amazing!"
}

Response

{
  "success": true,
  "data": {
    "text": "I really love this product, it is amazing!",
    "sentiment": "positive",
    "score": 0.85,
    "confidence": 0.85,
    "details": {
      "positive_words": 2,
      "negative_words": 0,
      "total_words": 8
    }
  }
}
{
  "success": false,
  "error": {
    "message": "Text parameter is required",
    "code": "INVALID_PARAMETERS"
  }
}
